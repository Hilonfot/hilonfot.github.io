4. ERP / 商城查看物料剩余库存 按区域查询

接口地址：’ wh_showKc_layer.php?getStock=true&token=’.$token.’&number=BC.004.0681’
请求方式： get
响应对象： JSON
token生成规则：md5(‘wh_showKc_layer’.date(“Y-m-d H:i”).’#$@%!’)
参数校验： getStock为true,token正确,number为物料编码
触发描述：客户在商城添加商品进入购物车时或者客户下单时
